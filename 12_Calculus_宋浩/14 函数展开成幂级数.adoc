== 函数展开成幂级数

求和函数, 是 stem:[1+x+x^2 + x^3+... = 1/(1-x) ] +
而"函数展开成幂级数", 就是倒过来问: 已知一个函数是 stem:[ 1/(1-x) ], 它能展开成什么? 等于什么幂级数(即相加)?

并非所有的函数, 都能表示成幂级数.
