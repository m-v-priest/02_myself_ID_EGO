[.small]

|英国建立香港的目的
|按照殖民大臣 Lord Stanley的說法: 佔領香港”並非為了殖民, 而是為了外交, 商業和軍事目的”. +
1970年前的最著名的香港史學家 G.B.Endacott 寫到: 健康的貿易需要安定的環境, 肅清盗匪, 保證契約得到遵從, 並確保司法的公正. 中國人被認為無法提供這些條件. 所以必須由英國人來提供.

**香港主要是为贸易而建立的城市. 它保证了自由贸易.** 香港外来中国市场又便利, 因此最终吸引到了各色外国商人到来.

虽然两次鸦片战争后, *中国开放了多个条约口岸通商, 但香港仍然是最可靠的商业基地,* 因为这个殖民地能被英国在政治上全权控制. *香港提供了”条约口岸”所不能提供的安全, 及行动和居住自由.* 也因此, 不久后,许多英国企业就以香港为亚洲总部.

- 在1840年代中期, 香港的外国企业中, 有近1/4 为印度人所有.
- 香港没有正式的中央银行, 其工作由汇丰银行担当, 直至殖民地时期的最后岁月.

|香港的经济类型
|香港的经济发展, 得益于为支持殖民地的贸易, 而兴起的多种服务业, 包括: 银行, 保险, 航运, 造船, 船舶维修等.

在1845-1941年, 香港政府的财政收入中, 很大部分是来自鸦片贸易. 此地的重要商人, 全靠鴉片致富. 政府也依賴高地租, 和其他由鴉貿易造就的收入, 來維持運作。

== 一戰期間

*第一次世界大戰期間, 戰事沒有直接波及香港.* 戰爭也沒有為香港帶來嚴重的經濟威脅, 除了通貨膨漲和某些貨品受到管制外.

香港和大英帝國其他地方一樣, 也出一分力支援英國參戰. 華人業主同意開徵7%的"特別戰務差餉". 并且幾近25%的英籍男性自願參軍, 还有大量華工在法國的華工軍團中服役. +
*由於歐洲貿易撤出亞洲, 本地華資公司因而得益.* 並且由於殖民地內約四分之一的歐籍男人, 奔赴戰場前線, 華商得以大舉進入銀行和航運等这些從前由歐籍人把持的商業領域.

1919年7月, 第一次世界大戰正式結束 (1919年6月28日簽署<凡爾賽條約>).

== 两次世界大战期间

1922年簽訂的《華盛頓公約》, 限制了英國能用於保護其在亞洲商業利益的軍力, 因此英國的工業生產值和世界貿易份額 都下降.

英國政府擔心從殖民地進口的貨品, 會與英國本土貨競爭, 在1930年代前 仍阻撓殖民地的政府發展工業.

1934年, 此時香港製造的貨品, 如化妝品和皮革製品, 已取代進口貨. 而本地生產的手電筒, 也攻佔從前由英資、日資公司把持的東南亞市場. 英國官員更抱怨從香港進口的貨品太便宜, 促使英國和加拿大政府 對之開徵”進口關税”.

'''

== 二戰期間

*日本封鎖了上海和其他中國港口後, 到 1938年初, 中國對外貿易中, 有一半是改經香港進行的.* 中國銀行和支通銀行等 也把總部遷到了香港, 把此地變成了全中國的銀行匯兑中心.


華人企業家不但帶來了資金, 還把整座工廠遷來了, 有些工廠是生產軍用裝備的, 供中國抗戰所需. 這幫助香港建設了工業基礎, 有利於香港戰後經濟的復甦.

來自中國的難民大批湧入, 也令這個殖民地本已擁擠的居住環境更加惡化, 但它也造就了土地買賣的興旺, 使政府收入大增.

香港經濟也大大得益於中國的國共內戰. 南來避禍的中國工業家, 為香港工業帶來新活力.

到了 1945 年11月, 經濟業已復甦得相當理想, 政府管制被取消, 自由市場回復.

到了 1946年初, 人口已回升至約100萬. 雖然這是與中國局勢混亂, 令移民穩定湧入有關. 貿易已將近回復到戰前水平的60%.



香港在戰後迅速復甦,  有助恢復港人對英國統治的信心. 尤其是相比較於日本佔領時期的苦難,  以及戰後中國的政治動盪和經濟凋敝.

國共內戰下, 國軍日衰. 從1947年至1949年,  上海主要企業幾乎全把業務遷移到香港, 令此地擁有了更大和更先進的工廠.

有些英國領袖, 從前曾懷疑香港的經濟價值, 但經過這場戰爭後, 對版圖日益縮小的大英帝國來說, 香港就顯得更加重要. 由於戰後"外國在華的租界"取消了, 令外國人多年來所享有的經濟安全,受到威脅, 許多外資公司不久就把總部, 由中國遷到香港. 香港因此成為英國在華商業樞紐, 戰略價值比以前更大.

== 冷战时期

==== 香港对中国的作用

如果中國願意的話, 其實可以更早收回香港. 但保留香港, 對中共有利: 英國治理下的香港, 能擔當中國通向外部世界的窗口.

- 中国因韩战被封锁后, 它能继续利用香港, 来获取 1.內地所必需的貨物, 通过走私 (天然气, 煤油, 盘尼西林等). 許多新界居民走私貨物到中國, 從"禁運"中獲利「所有能在中國賣得出去的東西, 都用貨車或船走私進去. 」2. 僑匯也經香港匯回中國.

- 从大躍進發生, 直至文化大革命, 中国一直在 **透過香港取得外匯,** 来做为購買進口貨的資金來源. 在1960年代, 中國近一半的硬貨幣, 是靠大陸向香港出售糧食和飲用水来賺取的.

== #★香港的经济转型: 从轉口貿易 -> 制造业 -> 金融业#

==== 从 轉口貿易 -> 制造业

[.small]
[options="autowidth" cols="1a,1a"]
|===
|Header 1 |Header 2

|香港经济转型的原因:
|50年代, 韩战期间 (1950-1953), 香港經濟因為兩個"禁運"而受害： +
(1) 聯合國以中國介入韓戰為由, 對中國實施的戰略物資禁運. +
(2) 美國對中國實施的"全面貿易禁運". 英國政府也限制出口到中國和香港的貨物.  (相當於香港的"進口量"和"出口量", 都被限制了)

*#韓戰中, 美國和聯合國的禁運, 迫使香港被逼步上從"轉口貿易"改為向"製造業"轉型.#* 如果沒有禁運, 香港在戰後可能不會達到如此高度的經濟繁榮 (*被迫产业升级, 人的进步都是被逼出来的.* 猶如中國被卡脖子, 就不得不自主發展芯片科技).

- 在1953年, 香港出口貨物中, 只有30%是本地製造. 到了 1959年, 這個比例升至70%. +
- 1960年代初, 香港已以"輕工業"產品(尤其是電晶體收音機等電子產品) 聞名於世.

|香港经济转型的推动者:
|不過, 這種以"輕工業"為重點的新發展, 並非由於殖民地政府採取什麼措施所促成的. 事實上政府仍然以狐疑的態度看待工業, 反而繼續強調香港的"貿易港"地位. 事實上**二次大戰過後, 殖民地政府長期不願鼓勵工業化, 因為這牴觸官方"着重貿易和商業"的殖民地政策.** 因此, 自由貿易和自由經營,「並不一定代表公平貿易和機會平等」. 這兩者是兩回事.

而率先引領香港轉型的, 则是本地華人企業家(雖然在禁運時期, 走私貨物進中國利潤很高, 但他們知道這種勾當風險很高, 也非長久之計）

促成香港經濟發展的因素, 也包括來自中國"穩定而廉價的勞動力"供應.
|===

'''

==== 从 制造业 -> 金融业

史學家一般認為,  *香港是在1960年代末至1970年初始, 掘起成為區內的金融中心.*

由1970年代開始, 香港與廣東的經濟聯繫越來越緊密. 香港提供"資金" 及"廣泛國際聯繫", 而廣東則提供廉價的勞動力, 雙方構成了經濟合作上的共生關係. +
香港投資者, 將大量資金投入到廣東, 設立工廠, 尤其是在1982年設立的"深圳經濟特區"中.

- 至1980年代末, 有超過300萬##**大陸人為廣東的"港資公司"打工. 這有助香港轉型, 從原來的製造業基地(生產電子產品等輕工業製品), 轉變成為領先的金融和服務業中心. **##

- 在天安門事件發生後的1990年代初, 香港製造商在中國政府鼓勵下, 將更多生產線遷往廣東省（在1979年, 香港有近90萬人從事製造業；到1997年, 這個數字降至38萬).  +
大型港資企業, 也前往中國投資房地產和基礎建設.

- **至1990年代中期, 約九成的香港工廠,已遷往中國, "製造業"佔香港本地蠱總值不足10%**（這種"生產環節"轉移到中國的趨勢, 對香港產生了重要的社會經濟影響. 有些技能變得再無用武之地, *而受衝擊最大的是藍領女工, 因為男人較容易在大陸找到管理職位*）. +
香港此時成為中國的最大投資者. 在1986年, 中國是香港第三大投資者（排在美國和日本之後）；到了1994年, 它在香港的投資, 相當於美國和日本加起來的兩倍以上.

- 在1970年代末, "服務業"佔香港本地生產總值, 不足65%; 到了 1990年代中期, 已升至差不多85%.

== 97后

*美國國會在1992年制定的 《美國—香港政策法》訂明, 只要美國總統認為香港有「足夠的自治」,  美國在九七後就繼續把香港當作與中國分開的領土看待.* 此法案是由肯塔基州共和黨參議員麥康奈爾(Mitch McConnell)提出, 因此也稱《麥康奈爾法案》(McConnell Act).

香港市民對於九七後前景的信心不斷下跌, 引發香港金融市場恐慌, 和房地產市場崩潰. 由於擊資者信心低迷, 香港的GDP出現自1975年以來最緩慢的增長.

- 在1983年9月, 跌勢不止的港元與美元掛鈎, 但這種信心沒有維持很久.
- 到了 1984年3月, 當時香港歷史最悠久和最龐大的英資公司怡和集團, 宣佈將註冊地點由香港遷往百慕達, 以求在 1997年後其業務運作可受英國法律保障.
