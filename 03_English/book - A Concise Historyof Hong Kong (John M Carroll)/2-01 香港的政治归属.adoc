== 1842 清廷被开放贸易 前

歐中的聯繫, 在唐朝(618-907)和元朝時, 歐洲人來中國通商是通過絲綢之路. 但元朝在14世紀中期土崩瓦解後, 中國就很少和歐洲接觸了.

歐洲在全球的擴張, 始自15世紀末, 一直持續到第一次世界大戰後.

西方商人, 在16世紀初才重返中國. 那時葡萄牙人開始在華南沿海通商.   +
葡萄牙人在1557時, 獲得明朝准許, 在澳門建立了永久殖民地 (perpetual 长期的，永恒的,（投资）永不还本的 colonial rights). 明朝的目的是: 1.回報他們協助清剿海盜. 2.將他們限制在特定區域.

雖然明朝禁止海外貿易, 但澳門不久就成爲了北半球的商贸交易中心.  同時, *澳門也成爲基督教和西學引入中國的基地. (類似於"長崎出島 Deshima"的荷蘭學, 對日本的影響)*


[.my1]
.案例
====
Dejima 出岛:

是位于日本长崎附近的一座人工岛 ，曾先后作为葡萄牙 （1570–1639）和荷兰 （1641–1858）的贸易站 。
 +
两百年来，荷兰人被禁止踏足日本。他们被限制在"出岛"上，生活条件极其恶劣。他们的贸易活动也受到严密监视。**在长达 220 年的时间里，出岛是日本闭关锁国时期（江户时代 ，1600–1869）**对外贸易和文化交流的中心通道，也**是当时唯一对西方开放的日本领土。** +
直到 19 世纪中期，荷兰人是唯一能够进入日本市场的西方人。

出岛在日本的**“兰学”（ rangaku ， 即荷兰语学习）** 运动中扮演了关键角色。"兰学"是一项学术活动，*旨在学习荷兰语， 以便了解西方的科学、医学和技术。*

1858 年，荷兰与日本签订条约 ，结束了出岛作为专属贸易站的地位，使其重要性大大降低。 +
最终，出岛通过填海造地, 并入长崎市。

到了1660年代, 英国开始挑战葡萄牙和荷兰, 力求成为在亚洲的贸易强国.
*几乎在英国势力出现在亚洲起, 它就有意在中国沿海建立可靠的贸易基地.* 因为当时荷兰, 葡萄牙和西班牙这些老牌海洋帝国, 都已经分别在巴达维亚(今雅加达), 澳门和马尼拉, 设有了据点. 而相较之下, 英国除了印度外, 在亚洲没有重要的领土.

1759年, 清政府宣布, 广州是中国唯一的对外通商口岸. 且限制西方人只能在每年10月至明年3月, 在廣州從事買賣. 其餘的每年4-9月, 西方商人必須返回澳門貿易.

18世纪末, 英中之间的贸易处在很不对等状态, 清廷只需要英国的白银, 而对英国的其他商品则没有什么需求. 为了扭转贸易不平衡, 英国人最后找到了将鸦片卖给中国. 中国对鸦片并不陌生, 之前, 鸦片在中国一直是用于医疗的. 但到了18世纪, 它就被用来做毒品吸食了. +
随着清廷对鸦片需求日益增加, 英国商人也愈来愈不满广州贸易制度的种种限制.

== 英国迫使中国开放贸易交流 → 南京条约 (1842)

之前, 英国在华贸易是被"东印度公司"垄断的. 但在1834年, 这种垄断权被英国政府撤销. 英国派 William John Napier (内皮尔 /威廉·约翰·律劳卑)来监督英国对华贸易.  +
Napier跟东印度公司的商人不同, 他不愿意跟广东海关打交道, 更坚决以外交官的身份, 同清朝中央官吏直接打交道. 但在清廷看来, 在外交上承认英国是与其平起平坐的国家, 清廷在亚洲的威望(朝贡宗主国) 将扫地.

*英国打第一次鸦片战争(1839年秋) 的目的, 就在于: 1.通商权力, 2.外交代表权.*

战后, <南京条约>签订(1842.8.29), 规定:


[.small]
[options="autowidth" cols="1a,1a"]
|===
|Header 1 |Header 2

|经济上:
|-中国结束"公行垄断制度" (对外贸易不再被清廷垄断, 可开放给民间) +
- 双方协定关税 +
- 开放五个口岸, 给于外国人通商和居住 +
- *英国享有”最惠国”待遇. 并保证日后中国赋予其他国家的优惠, 英国可一体均沾.*

|政治上:
|-香港岛(而非现在的整个香港范围), ”永久”割让给英国. +
- 英国享有”治外法权”.
|===

英國首次佔領香港, 是在1839年, 還處在第一次鴉片戰爭交戰期間. 那時沒幾個英國官員想將香港變成永久殖民地. 直到1843.6.26, 香港才正式成為殖民地. +
英国看中香港的原因, 是因为香港是个良港, 周围有山脉遮蔽, 不惧台风.

== 英国让中国进一步开放经济 → 天津条约 (1858)


对英国来说, <南京条约> 存在不足:

[.small]
[options="autowidth" cols="1a,1a"]
|===
|Header 1 |Header 2

|政治上:
|- 根据<南京条约>英文本, 英国人可进入广州城. 但中文版本, 却限制他们只能住在港口.
- 该条约没有赋予英国外交人员驻北京的权力. 英国官员也无法直接与广州的中国官员会面.
- #*条约没有订明, 日后可以修约.*# *而美国和中国签订的<望厦条约>(1844) 则规定了12年后可以修约. 所以, #英国希望援引<南京条约>的”最惠国条款”要求来修约 (即能享受和美国同样的待遇, 可以修约).#*
|===

因此, 英国人在1854年时, 就要求更多的权益, 包括外交官员驻节北京, 及在长江上自由航行 和从事贸易.

英,法,美,俄, 与中国签订<天津条约>(1858.6月), 规定:

[.small]
[options="autowidth" cols="1a,1a"]
|===
|Header 1 |Header 2

|经济上:
|-再开放10个通商口岸.

|政治上:
|-*允许西方外交人员驻节北京* +
- *让西方传教士和商人, 进入中国内陆腹地.*
|===



又签订<通商章程善后条约>(1858.10月), 规定: +
- 对进口鸦片征税. 实际上将鸦片贸易合法化了.
