== (港英) 香港的憲政架構

=== (港英)政府架構

image:/img/svg001.svg[,100%]

根據1843年4月5日頒佈的《英王制誥》(Letters Patent), 香港的管理模式是:


[.small]
[options="autowidth" cols="1a,1a"]
|===
|Header 1 |香港的政治架構、權力運作

|殖民地時期（19世紀至1941年前）
|香港並非三權分立，而是「行政主導」的殖民體制. *雖然表面上有行政、立法、司法三部分，但殖民地時期香港並沒有真正的「三權分立」. "立法局"也不是独立的"议会".* 整個政治架構是以"港督"（Governor）為中心的金字塔式行政統治體制.

港督同時是： +
- 行政首腦（政府首長） +
- 立法局主席 （直至1985年前. 因为1985年的香港政治制度改革, 建议港督不再担任"立法局"主席.） +
- 軍隊統帥 +
- 司法任命權掌握者 +

所以**三權最終都集中於"港督"手中，只是名義上分設不同機構。**

|1841–1941 日占时期
|1941年12月 , 太平洋战争爆发，香港沦陷，被日本占领（1941–1945）。

|1945–1980s 战后重建期
|1945年复殖后，政制虽仍非"三权分立"，但逐步**出现"本地化"倾向**。 +
仍是港督主导，但**"行政局/立法局"逐渐纳入华人**"非官守议员".

|1980s–1997 交还中国前的"过渡改革期"
|逐步引入"*有限选举*"，出现"直选"议席（1991年）. +
中英谈判后，*港英政府才在1985年起引入"间接选举"（functional constituencies），
1991年起才首次"直选"。*

1985年："立法局"进行第一次重大宪制改革##**（引入 functional constituencies 的"间接选举"），标志着"立法局"由"完全委任"转向"半代表制"**##。港督仍为法律上的主席，但"议会独立性"的象征性提升开始。

=== (港英) 香港的法律和司法體制

早期香港司法制度（特別是19世紀初期）並非如同英國本土那樣真正「三權分立」，而是更接近一種殖民行政主導下的「半司法制度」。


==== 法律

隨着愈來愈多領土 納入版圖, 大英帝國繼承了一張法律制度的大網, 這張網結合了英國普通法 、英國成文法, 和地方的習慣法.

吸納”地方法律”的部分原因, 是出於實際需要： +
1.英國人知道, 在一些像印度那樣幅員廣大和多元化的地方, 他們不可能猝然厲行根本變革. +
2. 英國人有一種看法, 認為管治"非英國人", 只能用他們的本土制度. +

同样, 香港的司法实际操作, 也不是依照英國的法律来的. (AI: 因为**香港是作為一個獨立的司法管轄區的.** 香港會從英國議會, 選擇性地採納少數適用於香港的成文法，但**大部分法律都是本地制定的。** 即, 雖然"普通法"的原則來自英國，但大部分的成文法（法律條文）都是本地化的。)

什么是**“帝国特惠制”（Imperial Preference）**？ 即: *成员之间相互给予关税优惠（preferential tariffs），而对帝国外的国家（尤其是美国、欧洲）维持或提高关税壁垒。* 即, 鼓励帝国成员内部互通有无, 保护帝国商品不被外国产品竞争.

会议后签署了一系列双边“互惠协定”： +
- 英国降低对"帝国内"农产品与原料的"进口关税"； +
- 殖民地与自治领, 则对英国制成品给予"关税优惠"； +
- 英镑区（Sterling Area）成员, 使用相同货币体系，形成封闭经济圈； +
- 贸易结算多使用英镑，有利于伦敦保持金融中心地位。 +

香港如何“受惠”? +
香港主要靠转口贸易与航运金融服务，并非靠出口原料或制造业. 当世界各地贸易萎缩时，香港的贸易渠道反而被“帝国内部”优先使用. +
*#帝國特惠制, 允許香港產品在全球經濟"貿易保護主義"浪潮中，仍然能憑藉"關稅優惠"進入廣闊的英帝國市場，使香港的經濟受損程度, 比許多其他依賴出口的經濟體要小得多。#*

二战后，帝国体系瓦解，1947年《关税与贸易总协定》（GATT）生效后，这种“帝国内部特惠”被逐步取消。

.香港政府沒有更盡力照顧大陸來港難民的原因是:
- 它認為一旦中國局勢穩定, 這些人就會返回中國.  +
- 許多殖民地官員擔心, 提供太多社會福利, 會吸引更多大陸難民來港.
- 安德葛說：「沒有人逼亞裔人(尤其是中國人)來港. 他們來了以後無論境況如何都得(自願)接受. 」

|60年代
|香港人口自1960年以來, 由約400萬躍升至近500萬. 這些新增人口, 大部分是在釀成巨災的##**中國"大躍進"（1958年至1961年）, 和其後的"三年饑荒"時期來港.**##  +

據估計單單在1962年就有約15萬名難民湧入.
1962年5月, 約6-10萬內地人, 在中國邊防守軍協助下, 非法越過邊界（雖然一個月內約6萬人被捕和遣返）. 香港警察和英軍其後在英方界線後, 築建起鐵絲網圍欄.
|本地身份認同, 其實早在十九世紀末已經出現, 那時候許多在港的富有華人, 開始視自己為此地的永久居民, 並且是一群特殊的華人, 不同於他們在中國大陸的同胞.  +

不過, 人口中其他階層的人, 則要待到1960和1970年代, 才開始對香港產生更加强烈的認同感.

大部分1950 年後在香港出生的人, 對中國沒有什麼經驗, 父母又是避禍南來的, 所以他們不大認同中華人民共和國, 特別是其共產黨政府. *香港人把香港的政治穩定和經濟自由, 對照於內地的壓迫和混亂, 尤其是在大躍進和文化大革命時期.*


|1975年越南難民 移入
|1975年**越南共產黨得勝統一全國後, 約七萬名越南船民為避難, 逃到香港.**
|但他們的出現也挑起恨意和敵意, 特別是本地的社會服務和房屋設施原已不足.

港府為遏止更多難民來港 , 在1982年7月實施禁閉營政策, 把大量越南難民, 關閉在由"懲教署"管理的禁閉營內. 政府並竭力安排其他收容國收留這些難民, 又勸他們自願返回越南. 但禁閉營一直存在至1992年.

|80年代
|
|港督麥理浩在1980年10月宣佈, 政府將取消「抵壘」政策.  +

#*Touch Base Policy 抵垒政策* : 是1974年11月-1980年10月间, 香港政府针对来自中国非法入境者的难民政策, **即由中国偷渡到香港市区（界限街以南）即得到香港居民的身份。*# *抵垒（Touch Base）本是棒球用语，*意指跑垒员只要能及时抵达并触碰垒包，便不会出局。*

*港府自1950年以來, 一直容許那些已經與家人團聚, 或覓得有酬工作的非法入境者, 留在香港. 但##現在"抵垒政策"取消後, 被捕獲的非法入境者, 則會被遣返.##*  麥理浩指, 這些非法入境者, 令房屋、醫療 、 教育和社會福利資源, 捉襟見肘. 而且這些人犯罪數目之多, 「與他們的人數完全不成比例」.

**1950年至1980年代，因为中国经历多次饥荒，大量人口偷渡香港，展开了为时三十多年的逃港潮。**1962年英属香港政府便大量遣返偷渡客。 +

1960年代以后，香港经济起飞，市场上有廉价劳动力的需求，香港政府便以黑印以及绿印身份证, 来区份"合法"以及"非法"移民，当时偷渡到香港的人口, 尚没有一个正式的香港居民身份.

|抵垒政策開始 (1974年11月)
|最终于##**1974年11月，香港政府宣布实施"抵垒政策": 中国"非法入境者"若在偷渡到香港后能抵达市区，便可在香港居留，如果偷渡者在边境范围被执法人员截获，则会被遣返。**##香港当局亦因此加强在边境的管制，以减少非法入境者进入市区的机会。

不过，港府對"抵垒政策"的心態, 亦是充满矛盾的: +

-> 實施"抵垒政策"的好處:

- **假如不给予抵垒者一个融入社会的途径，他们会沦为黑市居民或黑市劳工，有一部分人甚至或者会从事非法活动。**
- 给予他们合法居留, 也可纾缓当时需求殷切的劳动市场. 因为当时香港的工业，尤其是制衣业這类劳工密集型的行业, 能吸纳大批劳动力. 当中不少移民仍可自食其力，当中很多亦不须领用综援等社会福利开支。

-> 實施"抵垒政策"的壞處:

- *长期的"抵垒政策", 會激勵中国居民不断尝试冒险前往香港。*
- 组织偷渡的蛇头成为一种新兴的高利润行业.

|"抵垒政策"取消 (1980年10月23日)
|*转而采取"即捕即解"政策。所有非法入境者一经发现，即须被遣返中国。* 當時規定: 香港雇主雇用偷渡客者，可罚款5万港币并判监一年。
