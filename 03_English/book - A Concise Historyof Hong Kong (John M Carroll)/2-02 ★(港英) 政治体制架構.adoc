== (港英)政府架構

image:/img/svg001.svg[,100%]

根據1843年4月5日頒佈的《英王制誥》(Letters Patent), 香港的管理模式是:

殖民地時期香港並沒有真正的「三權分立」. 而是以港督为主导的. 港督同时掌控: +
① 行政权(是政府首長). 可任免地方官員;   +
② 立法权(是立法局主席, 一直到1985年为止. 可頒布本地法令, 但需報"殖民地部"備案或批准 );  +
③ 军权 (是軍隊統帥). 所以三權最終都集中於"港督"手中.

=== 總督 Governor of Hong Kong

[.small]
[options="autowidth" cols="1a,1a"]
|===
|Header 1 |Header 2

|港督的來源
|*在1970年代初以前, 香港总督都是由英国”殖民地部”官员出任. 他们来香港上任之前, 都曾在其他殖民地历练过.* 这些港督在其他殖民地的经验, 往往会影响他们对香港的态度.

殖民政府並非總能為所欲為, 它須向”殖民地部”, 和”英國國會”負責. 並經常須與大英帝國其他地方的法例, 保持一致.

|港督的職責和任務
|在1860年前, 香港總督身負三項獨特任務:

- 【外交】: 1.與中國當局談判,  ← 在《天津條約》和《北京條約》簽訂後, 令英國人獲得了外交權益, 英國"外交部"就接管對華事務, 港督從此就只專注于處理香港事務. +
- 【军事】: 2.保護英國在華貿易.  +
- 【经济】: 3.管理香港經濟. +

=== Executive Council 行政局（輔助決策）

[.small]
[options="autowidth" cols="1a,1a"]
|===
|Header 1 |Header 2

|成立時間：
|1843年

|性質：
|- 它只是港督的最高**##諮詢機構##**(相当于一个顾问班子)，協助港督制定政策、處理重大行政事務. **沒有獨立投票權，僅作諮詢。 ** 即, 儘管"行政局"名義上是決策機構，但港督擁有最終決定權。
- 實行「集體負責制」和「保密制」。





|成員：
|- *所有成員皆由總督任命。*
- 成員（當時稱為「議員」）分為：
.. 官守議員 (是高級官員)： 主要包括政府的三位司長（布政司、財政司、律政司）等，他們同時是行政部門的高層。
.. *非官守議員*（通常是商界代表組成）： 港督委任的社會賢達,或專業人士，*代表社會意見。*

|Executive Council ≠ Government Departments
|注意: 「行政局（Executive Council）」≠「行政機構（行政部門 Government Departments）」。
這兩者只是名字相似，但性質、功能、權力地位完全不同。

[.small]
[options="autowidth" cols="1a,1a,1a"]
!===
! !行政局 Executive Council!行政機構／政府部門 Government Departments

!是否具實權
!是"政策諮詢"機構, 無實際行政權. +

給港督提供政策建議、討論法案，但最終決策權在港督.
!是"行政執行"體系, 有實際行政權.  +
負責執行政策、管理具體事務.

!人员构成
!港督 + 官守議員（政府高官）+ 非官守議員（社會名流）
!各司、署、廳（例如：財政司、律政司、警察廳等）. 是龐大的公務員體系。
!===

比如港督要決定增加警察預算： +
他可能先在 “行政局” 會議上徵詢意見（例如財政司、律政司、非官守議員等人, 發表意見）。
但這只是討論性質，最終決定權仍在港督。
港督批准後，具體的執行就由”財政司”部門撥款、 ”警察廳”招人執行。

也就是說： +
行政局 ≈「港督的內閣顧問會議」 +
政府部門 ≈「港督所領導的行政執行機器」 +

雖然它们是兩個獨立的機構，但**它們通過「官守議員」緊密聯繫在一起**，體現了殖民地管治中「行政主導」的特點：  +
*"行政部門"中的最高層官員（如布政司、財政司、律政司）, 同時也兼任"行政局"中的官守議員。 这样, "行政部門"的官员想要做点什么, 就能通过"行政局"身份来獲得支持。(相当于"议会"中的"多数党"的政党领袖,担任"政府首相". 以让"政府"获得"议会"支持一样.)*

[.small]
[options="autowidth" cols="1a,1a"]
|===
|Header 1 |Header 2

|成立時間：
|1843年

|性質：
|- 是立法機構. 但**沒有"選舉制度"，無民意授權的議員, 所以也就沒有民意代表性。** (直到20世紀中期, 尤其是二戰後，英國才逐步讓香港人（尤其是華人）在這些機構中擁有”有限的代表性”。)

|人员构成
|- 官守議員（Official Members）：殖民地政府高官（如律政司、財政司等）。
- 非官守議員（Unofficial Members）：由總督委任的民間人士（多為商界代表，歐籍居多，後期才出現華人）。  +
大型英資企業的董事, 全都固定在"行政,立法"兩局擁有席位.

=== Judiciary 司法機構（審理案件）

香港採用英國普通法（Common Law）制度，但由於”行政主導”，實際上獨立性有限。

隨着愈來愈多領土 納入版圖, 大英帝國繼承了一張法律制度的大網, 這張網結合了英國普通法 、英國成文法, 和地方的習慣法.

吸納”地方法律”的部分原因, 是出於實際需要： +
- 英國人知道, 在一些像印度那樣幅員廣大和多元化的地方, 他們不可能猝然厲行根本變革. +
- 英國人有一種看法, 認為管治"非英國人", 只能用他們的本土制度. +

[.small]
[options="autowidth" cols="1a,1a"]
|===
|Header 1 |Header 2

|法官
|法官由港督任命，但通常來自英國本土法官團體。

|檢控官 (有"起诉"权)
|香港沒有"大陪審團", 所以大陪审团的职责, 就由"檢控官"来代理了. 检察官是由"律政司"來擔任的. *這意味著*”檢控權力”高度集中在”律政司”手中，*缺乏"民間對起訴程序的制衡"。*

(大陪审团的职责是什么? -- #*"大陪审团"决定“是否可以起诉”,*# 而不对"被告人是否有罪"做出最终裁决, 后者是"小陪审团" / Petty Jury 的工作。)

|律師
|英國政府在1857年, 才允准華人取得律師資格.

== 政府雇员, 公務員

[.small]
[options="autowidth" cols="1a,1a"]
|===
|Header 1 |Header 2

|歷史上一直 -> 不信任华人警员
|一直以来, 警员方面, 殖民地政府只聘用歐籍人, 而不聘用华人. 因为: +

- **政府擔心粤籍警察, 会無力和不願意管束他們的廣東同胞. **正如香港歷史早期時, *英國人常僱傭印度人為警察和獄卒, 因為他們不大可能協助華人囚犯逃走.*
- 他们认为,  歐籍的政府員工更可信賴, 對英國更忠心. +

因此香港政府只从從英國 、印度, 或威海衛, 招聘警察.



|二戰前大蕭條時期 -> 港府決定削減開支
|在經濟大蕭條期間, 香港沒有受到太大損害, 部分原因是香港受惠於 1932年”渥太華帝國經濟會議”後實行的”帝國特惠制”. 儘管如此, 英國和香港官員仍然擔憂殖民地的開支情況.

因为在1914年, 歐籍警員的薪金, 就已经比本地華籍或印籍警員, 高出逾七倍. 而到了 1939年, 低階歐籍警員的待遇, 幾乎是同級華人警察的八倍.
