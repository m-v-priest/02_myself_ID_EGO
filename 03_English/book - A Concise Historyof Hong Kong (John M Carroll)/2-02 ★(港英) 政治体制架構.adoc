== (港英)政府架構

image:/img/svg001.svg[,100%]

根據1843年4月5日頒佈的《英王制誥》(Letters Patent), 香港的管理模式是:

殖民地時期香港並沒有真正的「三權分立」. 而是以港督为主导的. 港督同时掌控: +
① 行政权(是政府首長). 可任免地方官員;   +
② 立法权(是立法局主席, 一直到1985年为止. 可頒布本地法令, 但需報"殖民地部"備案或批准 );  +
③ 军权 (是軍隊統帥). 所以三權最終都集中於"港督"手中.

|港督的來源
|*在1970年代初以前, 香港总督都是由英国”殖民地部”官员出任. 他们来香港上任之前, 都曾在其他殖民地历练过.* 这些港督在其他殖民地的经验, 往往会影响他们对香港的态度.

殖民政府並非總能為所欲為, 它須向”殖民地部”, 和”英國國會”負責. 並經常須與大英帝國其他地方的法例, 保持一致.

|港督的職責和任務
|在1860年前, 香港總督身負三項獨特任務:

- 【外交】: 1.與中國當局談判,  ← 在《天津條約》和《北京條約》簽訂後, 令英國人獲得了外交權益, 英國"外交部"就接管對華事務, 港督從此就只專注于處理香港事務. +
- 【军事】: 2.保護英國在華貿易.  +
- 【经济】: 3.管理香港經濟. +

=== Executive Council 行政局（輔助決策）

[.small]
[options="autowidth" cols="1a,1a"]
|===
|Header 1 |Header 2

|成立時間：
|1843年

|性質：
|- 它只是港督的最高**##諮詢機構##**(相当于一个顾问班子)，協助港督制定政策、處理重大行政事務. **沒有獨立投票權，僅作諮詢。 ** 即, 儘管"行政局"名義上是決策機構，但港督擁有最終決定權。
- 實行「集體負責制」和「保密制」。





|成員：
|- *所有成員皆由總督任命。*
- 成員（當時稱為「議員」）分為：
.. 官守議員 (是高級官員)： 主要包括政府的三位司長（布政司、財政司、律政司）等，他們同時是行政部門的高層。
.. *非官守議員*（通常是商界代表組成）： 港督委任的社會賢達,或專業人士，*代表社會意見。*

|Executive Council ≠ Government Departments
|注意: 「行政局（Executive Council）」≠「行政機構（行政部門 Government Departments）」。
這兩者只是名字相似，但性質、功能、權力地位完全不同。

[.small]
[options="autowidth" cols="1a,1a,1a"]
!===
! !行政局 Executive Council!行政機構／政府部門 Government Departments

!是否具實權
!是"政策諮詢"機構, 無實際行政權. +

給港督提供政策建議、討論法案，但最終決策權在港督.
!是"行政執行"體系, 有實際行政權.  +
負責執行政策、管理具體事務.

!人员构成
!港督 + 官守議員（政府高官）+ 非官守議員（社會名流）
!各司、署、廳（例如：財政司、律政司、警察廳等）. 是龐大的公務員體系。
!===

比如港督要決定增加警察預算： +
他可能先在 “行政局” 會議上徵詢意見（例如財政司、律政司、非官守議員等人, 發表意見）。
但這只是討論性質，最終決定權仍在港督。
港督批准後，具體的執行就由”財政司”部門撥款、 ”警察廳”招人執行。

也就是說： +
行政局 ≈「港督的內閣顧問會議」 +
政府部門 ≈「港督所領導的行政執行機器」 +

雖然它们是兩個獨立的機構，但**它們通過「官守議員」緊密聯繫在一起**，體現了殖民地管治中「行政主導」的特點：  +
*"行政部門"中的最高層官員（如布政司、財政司、律政司）, 同時也兼任"行政局"中的官守議員。 这样, "行政部門"的官员想要做点什么, 就能通过"行政局"身份来獲得支持。(相当于"议会"中的"多数党"的政党领袖,担任"政府首相". 以让"政府"获得"议会"支持一样.)*

=== Judiciary 司法機構（審理案件）

香港採用英國普通法（Common Law）制度，但由於”行政主導”，實際上獨立性有限。

隨着愈來愈多領土 納入版圖, 大英帝國繼承了一張法律制度的大網, 這張網結合了英國普通法 、英國成文法, 和地方的習慣法.

吸納”地方法律”的部分原因, 是出於實際需要： +
- 英國人知道, 在一些像印度那樣幅員廣大和多元化的地方, 他們不可能猝然厲行根本變革. +
- 英國人有一種看法, 認為管治"非英國人", 只能用他們的本土制度. +

[.small]
[options="autowidth" cols="1a,1a"]
|===
|Header 1 |Header 2

|法官
|法官由港督任命，但通常來自英國本土法官團體。

|檢控官 (有"起诉"权)
|香港沒有"大陪審團", 所以大陪审团的职责, 就由"檢控官"来代理了. 检察官是由"律政司"來擔任的. *這意味著*”檢控權力”高度集中在”律政司”手中，*缺乏"民間對起訴程序的制衡"。*

(大陪审团的职责是什么? -- #*"大陪审团"决定“是否可以起诉”,*# 而不对"被告人是否有罪"做出最终裁决, 后者是"小陪审团" / Petty Jury 的工作。)

|律師
|英國政府在1857年, 才允准華人取得律師資格.

== 政府雇员, 公務員, 警员

[.small]
[options="autowidth" cols="1a,1a,1a"]
|===
|Header 1 |公务员|警员

|歷史上一直 -> 不信任华人警员
|
|一直以来, 警员方面, 殖民地政府只聘用歐籍人, 而不聘用华人. 因为: +

- **政府擔心粤籍警察, 会無力和不願意管束他們的廣東同胞. **正如香港歷史早期時, *英國人常僱傭印度人為警察和獄卒, 因為他們不大可能協助華人囚犯逃走.*
- 他们认为,  歐籍的政府員工更可信賴, 對英國更忠心. +
- 要直到第二次世界大戰爆發前夕, 華人才獲准擔任”警察副幫辦”. 即便如此, 他們也得聽命於職位比他們低的英籍警察.

因此香港政府只从從英國 、印度, 或威海衛, 招聘警察.



|二戰前大蕭條時期 -> 港府決定削減開支
|在經濟大蕭條期間, 香港沒有受到太大損害, 部分原因是香港受惠於 1932年”渥太華帝國經濟會議”後實行的”帝國特惠制”. 儘管如此, 英國和香港官員仍然擔憂殖民地的開支情況.

为了"撙zǔn節用度"(即節省開支), 香港在公務員方面, 就開始聘任更多的非歐籍人. +
不过, 由于给華人員工开的薪水依然比歐籍人低, 缺乏吸引力, 所以增聘華人公務員的進度很緩慢.
|- 在1914年, 歐籍警員的薪金, 就已经比本地華籍或印籍警員, 高出逾七倍.
- 而到了 1939年, 低階歐籍警員的待遇, 幾乎是同級華人警察的八倍.

|二戰後
|- 在1950-1960年代, 香港获得一批”退役复出”的殖民地官员,公務員 -- 他们来自新进独立的英國殖民地和屬地 : 南亚,中东,和非洲.
- 直至1966年, 能擔任政府高層職位的華人, 仍是寥寥無幾.
|香港警隊在1960年代, 高級警察依然大多是来自外籍的資深警官 (來自馬來亞, 巴勒斯坦, 和非洲). 華人一般只是低級警察.
|===

==== British Empire Economic Conference 大英帝国经济会议 /渥太华会议 (1932):

.ChatGPT/gemini:
1929年美国股灾引发全球经济大萧条. 英国政府开始考虑：“能不能通过帝国内部的经济互惠，把市场‘关起来’，用帝国体系自救？” (內循環) +
会议在加拿大首都渥太华召开，出席者包括英国及其自治领（Dominions）——加拿大、澳大利亚、新西兰、南非、印度、爱尔兰自由邦等——的政府代表。

什么是**“帝国特惠制”（Imperial Preference）**？ 即: *成员之间相互给予关税优惠（preferential tariffs），而对帝国外的国家（尤其是美国、欧洲）维持或提高关税壁垒。* 即, 鼓励帝国成员内部互通有无, 保护帝国商品不被外国产品竞争.

会议后签署了一系列双边“互惠协定”： +
- 英国降低对"帝国内"农产品与原料的"进口关税"； +
- 殖民地与自治领, 则对英国制成品给予"关税优惠"； +
- 英镑区（Sterling Area）成员, 使用相同货币体系，形成封闭经济圈； +
- 贸易结算多使用英镑，有利于伦敦保持金融中心地位。 +

香港如何“受惠”? +
香港主要靠转口贸易与航运金融服务，并非靠出口原料或制造业. 当世界各地贸易萎缩时，香港的贸易渠道反而被“帝国内部”优先使用. +
*#帝國特惠制, 允許香港產品在全球經濟"貿易保護主義"浪潮中，仍然能憑藉"關稅優惠"進入廣闊的英帝國市場，使香港的經濟受損程度, 比許多其他依賴出口的經濟體要小得多。#*

二战后，帝国体系瓦解，1947年《关税与贸易总协定》（GATT）生效后，这种“帝国内部特惠”被逐步取消。

== 反腐與 /廉政


1949年后, 就有人建議設立"申訴專員", 調査有關政府部門行政失當的投訴, 但這個專責處理"行政投訴"的職位, 要待到1989年才設立.

到了 1970年代初, 貪污受賄已滲透在港府每一個部分, 上到高級官員, 下到警察(捲入集團式貪污).

#*1974年, 麥理浩开始建立”廉政公署”.*#  +
_Independent Commission_ （通常为政府管控或调查某事的）委员会 Against Corruption (Hong Kong)

|剛建立起的廉政公署, 面對的問題是:
|- 由於很多警察被捕, 警隊士氣大跌.
- 麥理浩擔心警察會罷工, 会令香港會陷入無政府狀態. 如果这样的话, 除非要求軍方来支援, 否則很難在短時間內招聘到足够他人, 來取代香港龐大警隊的职责.

|麥理浩的对策: 分而治之
|因此他頒佈”局部特赦令”: 除情節特別嚴重的案件外, 1977年1月1日前所犯的貪污罪, 一律不予追究. 目的是: (麥理浩後來憶述:)  +
1.**起訴那些在很久以前犯罪的人, 會令從前貪污, 和現在貪污的兩批人聯合起來. 而特赦能夠分裂這兩批人 (分而治之, 瓦解敵人的聯盟),**  +
2.簡化廉署的工作, 使它可以集中力量打擊當前的貪污罪行.
|===

"廉政原則"就成為了公務員文化的一部分.  香港市民可以有效投訴官員濫用職權, 玩忽職守, 及非法敲詐. 在香港当时特定的政治環境下(殖民統治和无民主), 此舉就令政府氣象一新, 成爲政府公信力的可貴來源.
