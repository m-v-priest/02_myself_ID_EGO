== 港英允许直选前

=== (1856) 当时港督提出: 增加一位选举的"立法局"议员 (失败)

政治代表權的問題, 很早就在香港歷史上出現了.  Sir John Bowring (香港第4任总督) 在1856年就建議, 向擁有土地, 並每年繳納超過十英鎊地租的人（包括華人）, 賦予”選舉一名**立法局**議員”的權利. 但並未成功.

- ”殖民地部”反對 Bowring 的主張, 理由是”代議政制”會令華人利益團體, 獲得支配地位, 且華人並未準備好實行”代議政制”.  +
雖然有些殖民地官員認為, 華人最好是由他們自己的「自然領袖」來管治, 但殖民地政府須小心行事, 以免給予華人領袖過多權力, 而招致歐人怨憤.
- 雖然本地歐籍商人, 要求以某種形式參與市政管治, 但他們堅決把華商排除在外. 這些歐籍人擔心, 提出這種要求"代議政制"的建議, 會令香港華人獲得過多權力, 這不但會損害英國的利益, 還可能幫助清廷控制香港.

|欧籍人的要求
|Constitutional Reform Association (政制改革協會) 在1894年陳情, 爭取權力:  +
- *1.有權選舉英國國民, 為香港"立法局"的”非官守議員”.* +
- *2.要求對殖民地財政, 有更大的控制權.* +

- 殖民地大臣Lord Ripon (曾任印度總督) 說, 他不能批准任何”排除華人的代議制度”. +

|结果与后续
|但反諷的是, 1894年這次爭取”政制改革”的要求, 其結果是華人的得益 大於歐籍人. 因為”*立法局*”增加了”官守”和”非官守議員”各一名.  而入選的非官守議員, 是個華人. +

1894年的”政制改革運動”后, 歐籍人此後沒有再認真重提”政制改革”, 直至第一次世界大戰後.

== 港英开始民选改革

=== 80年代的”代議政制”改革

*香港人雖享有高度的個人自由, #但在1980年代末和1990年代初之前, 香港沒有實行”代議制政府”, 也沒有政黨.#*

*在對香港的統治步入尾聲前, #一直拒絕實行"代議政制"的英國人, 在臨近1997的時候, 搖身一變, 忽然成為"民主改革"的堅定推動者.#* 而對比起中國政權, 這個形象更是鮮明. 有關殖民地政府的種族歧視, 和缺乏"政治代表性"的種種歷史, 消失殆盡.

1984年提議”代議政制”改革: +

- 在1984年7月, 香港政府發表諮詢文件《代議政制綠皮書》，改革方案包括: #**部分"立法局議員"由"間接選舉"選出.**#  +
- 雖然政府沒有承諾實行"直接選舉", 但它在1984年11月發表《代議政制白皮書》, 指: #在1988年逐步開始, **先"直接選出"很小部分的議員, 然後按次遞增,** 至1997年便**應有相當多的議員, 通過"直接選舉"選出」.**#

.案例
====

Wikipedia: +

這里面，#*with 的意思是： +
表示“以……为条件 / 依靠…… / 在……的情况下”*#

也就是说，这句话的结构是： +
Creating (v.) hopes（创造希望） +
for _universal suffrage_ and _democracy_（为了普选与民主） +
#*with*# _an equal and just Hong Kong_（在一个平等且公正的香港条件下 / 依靠一个平等公正的香港） +

##**因此 with 在这里**##不是“和……一起”那种简单的并列，而是##**更偏向： +
✔ “以一个平等、公正的香港作为基础” +
✔ “在拥有一个平等且公正的香港的情况下” +
✔ “借助 / 基于一个平等公正的香港” +**##

整句更自然的理解是：
*只有在一个平等与公正的香港，才可能为普选和民主创造希望。*


下面我给你一些 与 “with + 名词” 表示“在……条件下 / 以……为基础 / 因为有……” 的常见结构例句:

- With proper guidance, students can achieve (v.) amazing results.
→ 在有适当指导的情况下，学生可以取得惊人的成绩。
- With enough time, we can finish the project.
→ 有足够时间的话，我们就能完成这个项目。
