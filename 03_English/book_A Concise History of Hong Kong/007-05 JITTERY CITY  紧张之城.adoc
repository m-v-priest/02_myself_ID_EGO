=== 港币采用了 Linked Exchange Rate System (联系汇率制度)


Linked Exchange Rate System 联系汇率制度:

简单来说，**#联系汇率制度（Linked Exchange Rate System）就是一种“挂钩”协议：香港政府向全世界承诺，每 7.8 港元永远可以兑换成 1 美元。#** +
*为了实现这个承诺，##香港放弃了独立制定"利率"(注意不是"汇率")的权力，##选择让港元成为美元的“影子”。*

#*三大发钞行，每发行 7.8 港元，必须向香港金融管理局（金管局）交付 1 美元，换取一张“负债证明书”。这意味着，流通中的每一分港元，背后都有足额的美元实物顶着。*#

|强弱兑换保证：
|#*金管局设定了一个窄区间（7.75 - 7.85）。*#

- 强方兑换保证 (7.75)： *如果港元太抢手，升到 7.75，金管局就会卖出港元(加大对市场的供货)、买入美元，把汇率压回来。*
- 弱方兑换保证 (7.85)： *如果大家都在抛售港元，跌到 7.85，金管局就会动用美元储备买回港元(即向市场增加供应美元)，保住港币的汇率。*

|Impossible Trinity 三元悖论
|在经济学中，有一个著名的**“三元悖论” (Impossible Trinity)**。

*什么是“三元悖论”？一个国家或地区在金融政策上，无法同时实现以下三个目标，只能三选二：*

- #资本自由流动#： 钱可以随便进出（香港必须有，因为是金融中心）。
- #汇率稳定#： 汇率固定（香港选了，挂钩 7.8 保持稳定）。
- #独立货币政策（自主制定利率）#： 自己决定利息高低（香港因此被迫放弃了这一条）。

|为什么是放弃“利率”而不是“汇率”？
|汇率是被“锁死”的： 香港已经明确宣布 7.8 挂钩美元，这就是它的既定政策。所以，它不是“放弃制定汇率”，而是**“主动选择了固定汇率”**。

利率是被“绑架”的： #*为了维持这个 7.8 的固定汇率，香港的利息必须跟着美国走。*#

例子： 如果美国美联储（Fed）加息到 5%，而香港利息保持在 1%。 +
结果： 所有人都会把港元换成美元,去存储（因为利息高）。这会导致港元被疯狂抛售，汇率就会跌破 7.85。 +
反击： 香港金管局为了保住 7.8，只能跟着美国加息，把利息也升到 5% 左右，把钱留住。

利率自动调节机制 (The Adjustment): +
当港元汇率触及 7.85（弱方保证） 时，金管局会从市场买入港元、卖出美元。 +
收紧银根： 市场上的港元变少了（银行结余减少）。 +
利率上升： 钱少了，借钱的成本（利率）自然就上去了。 +
回归平衡： 香港利率升到跟美国差不多，抛售港元的操作停止，汇率回到 7.8。 +
