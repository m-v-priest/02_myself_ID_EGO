总结:

[.my3]
[options="autowidth" cols="1a,1a,1a"]
|===
||lunar eclipse 月食 |solar eclipse 日食

|成因
|#*地球运行到太阳和月球之间，地球的阴影遮蔽了月球。*#
|#*月球运行到太阳和地球之间，月球的阴影, 投射在地球上。*#


|现象
|#*月球变暗，有时呈现红橙色（血月）。*#
|#*月球遮住太阳，白天变暗。*#

|可见性
|范围大，地球上夜晚的大部分地区都能看到。
|*范围小，只能在月球阴影落在地球上的特定区域, 才能看到。*

|发生时间
|只发生在满月（望月）时。
|只发生在新月（朔月）时。
|===

#*一句话: 日食发生在白天，月食发生在夜晚。*#

下图看 ①影子(攻受) 和 ②地球的白天一面还是晚上一面, 就行了:

- #*月球是攻, 地球是受, 月球的影子投在地球上 -> 地球向阳面(即白天的那半球)能看到 -> 日食.*#
- #*地球是攻, 月球是受, 地球的影子投在月球上 -> 地球背阴面(即晚上的那半球)能看到 -> 月食.*#
